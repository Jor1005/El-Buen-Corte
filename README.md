# El Buen Corte 🪚

### Carpintería Artesanal de Alta Gama

Este proyecto es una Landing Page premium diseñada para **El Buen Corte**, un taller de carpintería artesanal en San Juan de Lurigancho. La interfaz refleja la precisión, la calidez de la madera y la robustez del oficio mediante un diseño editorial, asimétrico y totalmente responsive.

---

## 🎨 Paleta de Identidad
- **Bosque Profundo:** `#354130` (Seriedad y naturaleza)
- **Crema Papel:** `#F9F7F2` (Elegancia y limpieza)
- **Naranja Óxido:** `Orange-800` (Pasión y detalle)
- **Blanco Puro:** `#FFFFFF` (Contraste y respiración)

---

## ✨ Características Principales

* **Hero Section:** División asimétrica con efectos de escala en imágenes y tipografía serif de gran formato.
* **Navegación Inteligente:** Navbar oculto que aparece dinámicamente mediante scroll con transiciones suaves.
* **Sección About:** Composición asimétrica que rompe la monotonía visual mediante el uso de cuadrículas descentradas.
* **Servicios:** Grid de tarjetas interactivas con overlays oscuros y efectos de elevación al hacer hover.
* **Contacto Directo:** Sección enfocada a la conversión inmediata mediante llamada telefónica y WhatsApp, integrando Google Maps con filtros estéticos.
* **Responsive Design:** Adaptación meticulosa para dispositivos móviles y tablets (especialmente optimizado para el punto crítico de los 950px).

---

## 🛠️ Stack Tecnológico

* **React.js:** Estructura de componentes funcional.
* **Tailwind CSS:** Estilizado mediante utilidades para un rendimiento óptimo.
* **Lucide React / Heroicons:** Para la iconografía minimalista.
* **Framer Motion (Opcional):** Para las transiciones de scroll y aparición de elementos.

## 🚀 Instalación y Uso

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/el-buen-corte.git](https://github.com/tu-usuario/el-buen-corte.git)
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Iniciar el entorno de desarrollo:**
    ```bash
    npm run dev
    ```
